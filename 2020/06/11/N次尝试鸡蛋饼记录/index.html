(做菜系列)N次尝试鸡蛋饼记录
发表于 2020-06-11| 更新于 2020-08-22|美食

因何而起：

  • 早饭平时都是买的玉米和鸡蛋吃的，因为买的地方距离公司要走点路，早上去不想走过去买，所以产生了做鸡蛋饼作为早餐的想法
  • 成本更低，制作简单

随心而动：

第一次的尝试（2020-06.05) 上班的一次遐想

  • 食材：

    1. 两个鸡蛋

    2. 香葱数根（老板送的<白嫖>o(￣▽￣)ブ）

    3. 面粉

  • 制作过程：

    由于是第一次尝试，工具不足，所以面粉和水的量是很随意放的，步骤如下：

    1. 放入两个鸡蛋，加水搅拌均匀

    2. 加面粉搅拌均匀，因为水放的过多，在意识流的操作下，我多次添加了面粉，在多次加入面粉，液体仍然很稀的情况下，我又赔进去一个鸡蛋，≧ ﹏ ≦

    3. 加入盐，加入切好的小葱，搅拌均匀，多次搅拌到面粉块状物减少

    4. 热一下锅，放入油，把搅拌好的液体倒入，铺平，然后翻过来，由于水放多了，我在这一步的操作极为艰难

    5. 尝一尝，总结一下，第一次的尝试基本是失败的，因为水放多了的原因，加上翻转比较麻烦，最后并没有形成完美的饼状物，万幸尝了一下还行，还不是血亏

第二次的尝试（2020.7.15）[不知道吃啥的中午]

  • 起因：最近想到一个好的创意，时间有点不够啊，所以第二次相对于第一次的时间隔得有点久了，想起来这个事情主要是实在不知道做啥吃得，简单又能填饱肚子，从急吃面条有点腻呀，第二次的吸取了上一次的教训，控制了水的量，果然是比上一次好了很多，唯一的区别是太热了，实在是懒得去买葱呀！😳

  • 记录：


  • 口感

    • 质感：是比上次有进步得，上次感觉没有太熟，不过面粉生吃好像没啥事情😄
    • 味道：没有香葱，味道十分十分的平淡━━(￣ー￣*|||━━

  • 下一次计划

    因为这两次都是保持在做一顿得量左右，下次得想想如何批量生产，保证早饭能持久供应一周得问题，是一次直接整波大得呢，还是一次分多个小批次，整出一堆呢 🤔 还要考虑下味道
